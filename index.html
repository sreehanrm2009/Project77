Console Web App

Visual Studio Code is a source-code editor made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git.
Users can change the theme, keyboard shortcuts, preferences, and install extensions that add additional functionality.

Console is a feature in the Chrome and Edge DevTools. To open the Console, right click on the screen and click Inspect. Then click on the Console button at the top of the DevTools. The shortcut for this is Ctrl + Shift + J. To print anything on the console, use the console.log(Whatever you want the console to display); function to display something in the console.
